# Emoji 與 Type 對照

| Emoji | Type | 使用場景 |
|---|---|---|
| ✨ | feat | 新功能 |
| 🐛 | fix | 修 bug |
| 📝 | docs | 文件、註解 |
| ♻️ | refactor | 重構，行為不變 |
| 🎨 | style | 格式、命名，不影響邏輯 |
| ⚡ | perf | 效能優化 |
| ✅ | test | 測試相關 |
| 🔧 | chore | 設定、工具、相依套件 |
| 🔥 | remove | 刪除程式碼或檔案 |
| 🚀 | deploy | 部署相關 |

## 常用動詞

| 動作 | 可用動詞 |
|---|---|
| 新增 | 新增、加入、建立 |
| 修改 | 更新、調整、修改 |
| 修復 | 修正、修復、解決 |
| 移除 | 移除、刪除 |
| 重構 | 重構、調整架構、整理 |
